<div><h2 id="promise-" style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin-right: 0px; margin-bottom: 16px; margin-left: 0px; padding: 0px 0px 0.3em; top: -900em; left: -900em; box-sizing: border-box; border-bottom: 1px solid rgb(238, 238, 238); line-height: 1.25; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;; margin-top: 0px !important;">Promise 的含义</h2><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;">Promise 是异步编程的一种解决方案，比传统的解决方案——回调函数和事件——更合理和更强大。它由社区最早提出和实现，ES6将其写进了语言标准，统一了用法，原生提供了<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;">所谓<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>，简单说就是一个容器，里面保存着某个未来才会结束的事件（通常是一个异步操作）的结果。从语法上说，Promise 是一个对象，从它可以获取异步操作的消息。Promise 提供统一的 API，各种异步操作都可以用同样的方法进行处理。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;"><code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象有以下两个特点。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;">（1）对象的状态不受外界影响。<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象代表一个异步操作，有三种状态：<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Pending</code>（进行中）、<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Resolved</code>（已完成，又称 Fulfilled）和<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Rejected</code>（已失败）。只有异步操作的结果，可以决定当前是哪一种状态，任何其他操作都无法改变这个状态。这也是<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>这个名字的由来，它的英语意思就是“承诺”，表示其他手段无法改变。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;">（2）一旦状态改变，就不会再变，任何时候都可以得到这个结果。<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象的状态改变，只有两种可能：从<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Pending</code>变为<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Resolved</code>和从<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Pending</code>变为<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Rejected</code>。只要这两种情况发生，状态就凝固了，不会再变了，会一直保持这个结果。如果改变已经发生了，你再对<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象添加回调函数，也会立即得到这个结果。这与事件（Event）完全不同，事件的特点是，如果你错过了它，再去监听，是得不到结果的。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;">有了<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象，就可以将异步操作以同步操作的流程表达出来，避免了层层嵌套的回调函数。此外，<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>对象提供统一的接口，使得控制异步操作更加容易。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;"><code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>也有一些缺点。首先，无法取消<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>，一旦新建它就会立即执行，无法中途取消。其次，如果不设置回调函数，<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>内部抛出的错误，不会反应到外部。第三，当处于<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Pending</code>状态时，无法得知目前进展到哪一个阶段（刚刚开始还是即将完成）。</p><p style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 0px 0px 16px; padding: 0px; box-sizing: border-box; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, &quot;Segoe UI&quot;, Roboto, Helvetica, Arial, sans-serif, &quot;Apple Color Emoji&quot;, &quot;Segoe UI Emoji&quot;, &quot;Segoe UI Symbol&quot;;">如果某些事件不断地反复发生，一般来说，使用 stream 模式是比部署<code style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; box-sizing: border-box; font-family: Consolas, &quot;Liberation Mono&quot;, Menlo, Courier, monospace; padding: 0.2em 0px; margin: 0px; font-size: 12px; background-color: rgba(0, 0, 0, 0.04); border-radius: 3px; color: rgb(255, 102, 102);">Promise</code>更好的选择。</p><h2 id="-" style="text-rendering: optimizeLegibility; -webkit-font-smoothing: antialiased; margin: 24px 0px 16px; padding: 0px 0px 0.3em;
